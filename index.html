🔐 Sistema Avanzato di Gestione Stash

Questo script offre un sistema pulito e intuitivo per la gestione degli stash (depositi) nei server FiveM. Permette di creare stash personalizzati con configurazione di slot e limite di peso.

✅ Caratteristiche principali:

  • Nome stash personalizzabile
  • Configurazione degli slot
  • Impostazione del limite di peso
  • Controllo accesso basato su accesso pubblico o solo job

⚠️ L’accesso agli stash è disponibile esclusivamente in modalità pubblica o basata sui job.

🔧 Per abilitare l’accesso, aggiungi questa riga nella configurazione ACL del server:

add_ace license:xxxx stash.creator allow

✨ Realizzato per framework ESX (QBCore non ancora supportato).

🔐 Advanced Stash Management System

This script offers a clean and intuitive stash (storage) management system for FiveM servers. It allows you to create custom stashes with configurable slots and weight limits.

✅ Key features:

  • Custom stash name
  • Slot configuration
  • Weight limit setting
  • Access control based on public access or job roles only

⚠️ Warning: Access to stashes is available exclusively in public mode or job-based mode.

🔧 To enable access, add this line to your server’s ACL configuration:

add_ace license:xxxx stash.creator allow

✨ Built for ESX framework (QBCore is not yet supported).
